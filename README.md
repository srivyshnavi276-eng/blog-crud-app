# Blog CRUD Application (Task-2)

A Basic CRUD web application built using PHP + MySQL with User Authentication.

## Features
- User Registration & Login
- Password Hashing
- Session Management
- Create, Read, Update, Delete blog posts
- Each user can view/manage only their own posts
- Simple CSS UI

## Tech Stack
- PHP (PDO)
- MySQL
- XAMPP

## How to Run
1. Copy folder to `xampp/htdocs/blog-crud`
2. Start Apache and MySQL in XAMPP
3. Import / create DB `blog` and tables `users`, `posts`
4. Open: `http://localhost/blog-crud/login.php`
